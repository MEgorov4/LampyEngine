# Game Engine (C++ / OpenGL)

Учебный движок с DI-модульной архитектурой.  
Рендер на OpenGL, сборка через CMake + Conan.

- **ECS:** Flecs.
- **Физика:** Bullet3.
- **Скрипты:** Lua (sol2).
- **Ресурсы:** stb_image, tinyobjloader, nlohmann_json.
- **Прочее:** clip (буфер обмена), Boost.Process, GoogleTest.
